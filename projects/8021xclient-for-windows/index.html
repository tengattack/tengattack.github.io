802.1X客户端 for Windows

Tuesday, October 23, 2012 11:21 AM
用于取代iNode H3C的一个Windows程序。 项目页面 https://github.com/tengattack/8021xclient-for-windows 程序下载 https://github.com/tengattack/8021xclient-for-windows/raw/master/Release/xclient.exe fork from njit8021xclient: https://github.com/liuqun/njit8021xclient depends on WinPcap: http://www.winpcap.org/ 使用说明 1.要想运行本程序，请先安装WinPcap(http://www.winpcap.org/)，如果你是Windows8系统，请下载Windows7的安装包，右键属性设置兼容模式为Windows7，点开安装包，如果有提示，点击运行而不获取帮助。 2.本程序为命令行使用，基本运行方式为 xclient.exe [用户名] [密码] [网卡名] 其中网卡名可以不用输入，这样的话在用用户名和密码启动程序后会提示选择一个网卡，多网卡的话输入y回车，即为选择该网卡，直接回车为显示下一个网卡的信息。 网卡名请使用程序输出的网卡名，例如: \Device\NPF_{EC2AB8A3-4D81-43ED-B53C-673056F99829} 执行程序的一个例子 xclient.exe username password \Device\NPF_{EC2AB8A3-4D81-43ED-B53C-673056F99829} 3.更新Windows的IP地址，如果你是刚插上网线就使用本程序的话，Windows可以自动获取到IP。 除此之外，可以先随便修改一个IP，应用后再改为自动获取IP，这样的话Windows也可以自动获取IP。 另外，还可以尝试使用这个工具进行DHCP的更新: https://github.com/tengattack/8021xclient-for-windows/raw/master/SetDHCP/WMIAdapter.exe 使用命令行，如: WMIAdapter.exe {23D1E054-A1F9-44FD-8DCA-642EFE43A360} 后面的参数是网卡的GUID，可在xclient里列出的网卡名称中看到，只需要GUID即可。
